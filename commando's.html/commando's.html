commando's Git

  • Git init: is een commando om een nieuwe Git repository te gaan maken.
  • Git add: voegt een wijziging in de werkmap toe aan de staging area.
  • Git comit: momentopname van alle wijzigingen in je project op een specifiek moment.
  • Git status: toont de huidiga status van de Git repository
  • Git log: een git commando dat wordt gebruikt om de geschiedenis van alle commits in een repository te bekijken.
